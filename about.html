Project mission

Bellbrook Bridge Automated Floodwatch

This page supports a simple local objective: make the Bellbrook bridge gauge easier to read quickly on a phone.

Why it was built

This tool was built because the available official flood information was not presented in a clear, direct, bridge-first format for everyday local use. Bellbrook residents need a fast plain-language view of the gauge: where the water is, how far it is below the 2.80 m deck, and whether the water is approaching deck level.

What it does

The dashboard reads a published river-height feed, displays the latest gauge height, calculates clearance against the 2.80 m bridge deck reference, and shows a local crossing-status band.

What it is not

This dashboard does not replace emergency warnings or road-closure notices. It exists because flood information is fragmented across separate sources and is not always presented in one clear local view.

How to use it

Use the dashboard as a fast local visual check. During rain, rising water, debris, darkness, or poor visibility, compare the gauge reading, the live camera, road conditions, and any warnings available at the time before making decisions.

Design standard

The interface is high-contrast, phone-first, and designed for fast reading under stress. The key number is bridge clearance. If the reading reaches or exceeds the 2.80 m deck reference, treat the bridge as at deck contact.
